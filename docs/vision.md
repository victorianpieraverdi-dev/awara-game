## 0. Главная сверх-цель
Игра — инструмент **подготовки игрока и планетарного Логоса** к переходу на следующий вибрационный уровень. Рассчитана на 30 000 лет, фрактальная, неотделима от жизни. Финал — игрок выходит из игры с интегрированными принципами и **создаёт свою**.

## 1. Уровни эволюции игрока (ось)
1. **Инициация / душа** — текущий контур (ДУША · ОСНОВА · СЕРДЦЕ · МЕЧТА)
2. **Земля игрока** — Васту, дом, храм, стихии (земля/вода/огонь/воздух/эфир)
3. **Космос игрока** — звёздный храм, связь с космосом
4. **Карта мироздания / божественные игры** — игрок становится божественным игроком
5. **Светообмен** — отдача всем живым существам
6. **Свои игры**

Внутренняя мифо-ось: **ахамкара → душа → жива → искра → бытовой дух → планетарный дух → дух единого**.

Фрактал: каждый уровень включён в следующий. Прокачивается душа → окружение → земля → космос → божественная игра.

## 2. Светкоин — экономика по уровням (ключевая правка лора)
Чем ближе к источнику, тем **больше** света зарабатывается за единицу опыта (нулей **меньше**, не больше):

| Уровень | Прирост за событие |
|---|---|
| Инициация (текущий) | ~0.0000000001 ✦ (10⁻¹⁰) |
| Земля игрока | ~0.000001 ✦ (10⁻⁶) |
| Космос игрока | ~0.0001 ✦ (10⁻⁴) → 0.001 → 1 → 10... |
| Игра Мироздания | 0.01–0.1 ✦, в т.ч. **пассивно** через накопленные ключи/карты |

Свет вибрирует интенсивнее ближе к центру. Ошибок меньше — точных движений больше.

**Светкоин-банк / дашборд**: видно вклад каждого игрока, топ контрибьюторов.

## 3. Социальный слой
- **Чат игроков** — переписка, объединение, **отправка светкоинов**, лайки, комментарии
- Творчество в светообмене: фото/видео/аудио/файлы/статьи/услуги
- AI оценивает «стоимость» в светкоинах
- Соц-граф между игроками (объединения)

## 4. Натальная карта & персонализация
- Игрок вводит дату/место рождения
- Синхронизация с внешним источником натальных карт (TBD какой)
- Игра **подстраивается** под игрока: матрицы, культуры, советы, ритм
- Стихии конституции (доши): земля/вода/огонь/воздух/эфир — нарабатываются записями в сферах и Тигле
- Айурведа — питание, поведение, как себя вести
- Накшатры — матричные агенты (ведическая основа), Pavel пришлёт материалы

## 6. 33 матрицы культуры (визуальная трансформация)
- Игрок выбирает культуру/проходит через неё → **визуальный код игры подстраивается**: цвета, символы, типографика, элементы
- Pavel пришлёт справочник 33 культур и накшатр
- Возможно сократить число матриц
- Доступ: открывается по мере накопления светкоина и светообмена

## 7. Карта мироздания (новый уровень)
Интерактивная карта космоса + **географии реального мира**:
- Где игрок физически находится → юг/запад/север/восток → стихия
- Возвращается домой → отчёт, интеграция в виртуальную игру
- 3 уровня восприятия территории:
  - **Подсознательный** — что чувствует на территории (чувствование негативной/позитивной поляризации городов)
  - **Сознательный** — с кем/чем взаимодействует, узлы
  - **Сверхсознательный** — что приходит из духа/интуиции
- 62/64 фестивальные пространства по планетарному Логосу — точки сакральной энергии

## 8. Тигель — расширение

- Надо:

  - 7 параметров на выходе: энергия дня, энергия натальной карты, итог, что будет завтра, образ (визуал), музыка (Suno API?), айурведический совет

  - Распределение AI-агентом по: сферы инициации + Земля игрока + Космос игрока + карты
  - Образы: на тестовом этапе — Pavel шлёт руками, потом — генератор через API
- **Искра** в пространстве — летает по сферам, собирает информацию, даёт подсказки (вопрос: оставлять или нет?)
- **Оракул** — диалог о следующем уровне, как сгармонизировать текущие уровни
