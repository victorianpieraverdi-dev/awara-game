# AWARA Рефакторинг — активные задачи после аудита

Источник: `docs/REFACTOR_TASKS.md` и аудит текущего репозитория от 2026-05-16.

Правила для всех задач:

- Не пушить напрямую в `master`; только feature-ветка и PR.
- Один номер задачи — один коммит.
- Не трогать канон: 21 агентов, 33 матрицы, 14 лок, 9 чакр-мер.
- Не удалять legacy-код, пока новый модуль не подключён и не проверен.
- Для UI-изменений проверять mobile 320 / 375 / 768.
- Коммиты на русском, без декоративных символов.

## Почему новая нумерация начинается с T-050

- T-049 уже закрыта в текущем `TASKS.md` как `game-space-changes.md`.
- Старую T-050 из исходного плана удаляем: базовые папки `css/`, `js/`, `assets/`, `docs/`, `data/` уже существуют.
- Задачи по уже существующим экранам удаляем или оставляем только там, где функционально ещё нет отдельного файла.
