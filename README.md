# AWARA v258 - Модульная версия

Проект разбит на отдельные модули для удобства разработки и поддержки.

## Структура проекта

```
awara-v258-modular/
├── index.html          (11170 строк) - Основная HTML структура
├── css/
│   └── main.css        (1958 строк)  - Все стили игры
├── js/
│   ├── core.js         (660 строк)   - Основная логика игры
│   ├── ui.js           (396 строк)   - Интерфейс и UI элементы
│   ├── spheres.js      (124 строки)  - Работа со сферами
│   ├── init.js         (560 строк)   - Инициализация игры
│   └── mechanics.js    (541 строка)  - 7 механик (Трофеи, Дневник, Квесты, Оракул, Медитация, Карма, Казна)
└── data/               - Папка для конфигов и данных (пока пуста)
```

## Запуск

Просто откройте `index.html` в браузере. Все модули подключаются автоматически.

## Преимущества модульной структуры

- **Легче редактировать** - каждый файл отвечает за свою часть
- **Быстрее загружается** в редакторе
- **Проще отлаживать** - можно быстро найти нужный код
- **Удобнее работать в команде** - меньше конфликтов при слиянии
